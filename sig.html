JavaScript:

  • Lightweight, interpreted, or just-in-time compiled programming language.
  • Scripting language for Web pages.
  • For non-browser environment: Node.js, Apache CouchDB and Adobe Acrobat are used.
  • Prototype-based, multi-paradigm, dynamic language, supporting object-oriented, imperative, and declarative.

JavaScript Libraries and Frameworks:

Node.js

Node.js is a free open source server environment - a way to use JavaScript on the backend. It has a lot of supporting packages and libraries.

React

React is used to help create small- or large-scale web applications, and is particularly useful in creating interactive websites.

jQuery

jQuery is a library used primarily for Document Object Model (DOM) manipulation. It can select DOM elements, create animations, handle events, and more.

Bootstrap

Bootstrap is an open-source front-end framework to help you with designing websites. It makes it easy to create responsive designs.

Angular

AngularJS is a front-end framework that helps you create web applications. It uses directives that can be inserted into the HTML to give the application more functionality.

Vue.js

Vue.js is an open-source progressive JavaScript framework. Some of its features include components, templates, transitions, and two-way data binding, as well as a focus on reactivity.
